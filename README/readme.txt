目录结构
├─common 共公的数据
│ ├─extensions 第三方扩展目录
│ ├─services 基础服务
│ ├─config 配制文件
├─backend  普通管理员
│ ├─cms   模块名
│ │ ├─controllers 控制器目录
│ │ ├─models 模型目录
│ │ ├─views  视图
│ │ ├─service  服务层
│ │ ├─libs  封装的类
│ │ ├─utils 工具类
│ │ ├─cron  定时任务目录（可选）
├─super  超管
├─api  手机接口
├─data 上传文件目录
│ ├─goods 商品相册上传目录
│ │ ├─20171207
│ ├─goods_brand 商品品牌图片
│ ├─goods_category 商品分类图片
│ ├─goods_qrcode 商品二维码图片
│ ├─goods_sku 商品SKU图片
│ ├─qrcode 店铺二维码
│ ├─advertising 广告位
│ ├─avator 用户头像
│ ├─common 公共上传文件目录
